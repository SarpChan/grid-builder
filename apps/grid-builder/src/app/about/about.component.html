Thank you for using Grid Builder! This tool was created to help you design and generate CSS grid layouts.

Grid Builder is a free and open-source project built using Angular and Tailwind CSS. The code is available on Github. Feel free to contribute to the project or open an issue if you find a bug.

Please also visit my personal website sarp-can.com

Grid Builder does not phone home or make any request for that matter. Everything runs local, files you upload are not analyzed and do not leave your computer. Downloaded files are generated on the fly and are not stored.
